# Star-Battel
⭐ Star Battle Puzzle

Star Battle is a logic puzzle played on an N×N grid. The goal is to place K stars in each row, column, and region without any two stars touching, not even diagonally. It’s a challenging puzzle that tests reasoning and strategy.

This Java console application allows you to:

Represent the puzzle board and regions in ASCII format
Read puzzle definitions from text files
Validate user moves and provide instant feedback
Check if the board is solved correctly
Solve puzzles automatically using a basic strategy with step-by-step reasoning
💡 Features
Console-based, no GUI required
Clear, readable ASCII grid display
Move validation with informative messages
Step-by-step solving strategy for learning logic
Flexible input: supports custom board sizes and regions
📖 How It Works
Load a puzzle from a text file
Place stars manually using text commands or let the solver run
The program validates each move and updates the board
When all rules are satisfied, it confirms: “Puzzle solved!”
🛠️ Technologies
Language: Java
Text-based interface
File input/output for puzzle definitions
Modular, object-oriented code structure
🎯 Purpose

This project was created to practice:

Logic implementation and condition handling
Working with arrays and 2D data structures
Modular programming and clean code
File I/O and console-based interaction
